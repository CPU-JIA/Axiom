# 数据库备份目录

此目录用于存储数据库备份文件。

## 备份命名规则

备份文件按以下格式命名：
```
{数据库名}_{时间戳}.sql
例如: euclid_elements_20240722_143052.sql
```

## 使用方法

### 创建备份
```bash
make db-backup
```

### 恢复备份
```bash
make db-restore FILE=backups/euclid_elements_20240722_143052.sql
```

## 注意事项

- 备份文件包含完整的数据库结构和数据
- 恢复操作会覆盖现有数据，请谨慎操作
- 建议定期清理旧备份文件以节省空间
- 生产环境请配置自动备份策略
